# 需求
实现一个通用的数组类，要求：
1. 可以对内置数据以及自定义数据类型的数据进行存储
2. 将数组中的数据存储到`堆区`
3. 构造函数中可以传入数组的容量
4. 提供对应的拷贝构造函数以及`operator=`防止浅拷贝问题
5. 提供尾插法和删尾法对数组中的数据进行增加与删除
6. 可以通过下标的方式访问数组中的元素
7. 可以获取数组中当前元素个数和数组的容量
